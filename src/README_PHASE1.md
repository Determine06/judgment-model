This folder hosts code for the V3 script-based pipeline (Phase 1 structure).

- build: Orchestration and data/build-time scripts (ETL, preprocessing, indexing).
- inference: Runtime inference/serving scripts and batch generation entrypoints.
- common: Shared utilities, helpers, config loaders, and constants used across modules.
- prompts: Prompt templates, prompt JSON/TXT, and related small config artifacts.
